# AI Startup Idea Generator

## Project Overview

AI Startup Idea Generator is a Multi-Agent AI application built using Python, LangChain, Gemini API, and Streamlit.

The application helps users generate startup ideas and automatically performs:

- Market Research
- Competitor Analysis
- Business Model Generation
- Investor Pitch Creation

The final report can be downloaded as a professional PDF.

## AI Agents

### 1. Idea Generator Agent

Generates startup ideas based on the user domain.

### 2. Market Research Agent

Analyzes:

- Target Audience
- Market Demand
- Industry Trends
- Potential Customers

### 3. Competitor Analysis Agent

Analyzes:

- Competitors
- Market Gaps
- Business Opportunities

### 4. Business Model Agent

Generates:

- Revenue Model
- Pricing Strategy
- Monetization Plan
- Go-To-Market Strategy

### 5. Pitch Agent

Creates:

- Problem Statement
- Solution
- Value Proposition
- Elevator Pitch

## Installation

### Clone Repository

```bash
git clone <repository-link>
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create .env File

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

### Run Application

```bash
streamlit run streamlit_app.py
```
